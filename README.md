# Google Homepage Clone

A front-end practice project recreating the visual layout of the Google search homepage using HTML and CSS.

## What This Project Includes

- Top navigation bar with quick links (About, Store, Gmail, Images, Settings, Sign in)
- Centered Google logo and search bar with search icon and mic icon
- "Google Search" and "I'm Feeling Lucky" buttons
- Footer with region indicator and links (Advertising, Business, Privacy, Terms, etc.)

## Tech Stack

- HTML5
- CSS3 (custom styling, no framework)

## Purpose

This project was built purely as a **static layout/CSS practice exercise** to recreate a well-known UI from scratch. It does not perform any real search functionality — all buttons and links are placeholders (`#` or non-functional).

## ⚠️ Note on Hosting

This project intentionally mimics a real, well-known company's homepage (Google). Because of that, deploying it live (e.g. via GitHub Pages) can trigger browser phishing/safety warnings, since automated tools can't distinguish a harmless practice clone from an actual phishing page impersonating Google. This repo is meant to be viewed as source code / run locally, not hosted publicly as a live site.

## How to Run Locally

1. Clone this repository
2. Open `index.html` directly in your browser

No build steps, dependencies, or installations required.
